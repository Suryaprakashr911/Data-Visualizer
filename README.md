# Data---Visualizer
# Dynamic Multi-Type Chart Generator

This project is a web-based chart visualization tool that allows users to paste JSON data or upload CSV/JSON files to generate dynamic charts using [Chart.js](https://www.chartjs.org/). Users can choose between various chart types such as Pie, Line, Bar, Radar, Doughnut, and Polar Area.
## Features
- 🖼️ Supports multiple chart types: Pie, Bar, Line, Doughnut, Radar, Polar Area
- 📄 Accepts input as:
  - Pasted JSON data
  - Uploaded `.json` or `.csv` files
- 🔄 Dynamically updates the chart when a new type is selected
- ✅ Validates data input
- 🛠️ Easily extendable with additional chart types or themes
